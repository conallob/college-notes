Linear maps: formulae explained

Rationale

We are given a formula F(X) supposed to define a linear map in 2 dimensions, say.

Suppose that the formula has been shown to describe some linear map, but we want to show that it describes a specific linear map f (such as perpendicular projection onto a line OP).

We want to be certain thet the formula `works': i.e., that F(X) = f(X) for every point X in the plane.

It is enough to produce an ordered basis P,Q for the plane and show that F is `correct' on P and Q, i.e., that F(P) = f(P) and F(Q) = f(Q).

This is because F and f are both linear, and P and Q are a basis. Suppose that X is any point in the plane. There exist real numbers a and b (actually unique for X, though that isn't relevant here), such that

X = a P + b Q . Since F and f are both linear,

F(X) = a F(P) + b F(Q), and f(X) = a f(P) + b f(Q),

Therefore F(X) = f(X), which is what we wanted to prove.

Exactly the same idea will work in 3 dimensions, except that we shall need 3 points P, Q, and R to form a basis.
