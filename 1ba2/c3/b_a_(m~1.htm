Chapter 3: Variables, types, and assignment

  1. Say whether each of the following tokens is a Java keyword, a valid identifier, or a literal. In the case of a literal, say what the type of the literal is. In the case of an integer literal say what base the number is in and its value in decimal.
    myName while ELSE 5.0E3 0x56A1
    02547L numOf$ 2345.78F timeToGo timetogo
    1110 01110 8945.89e-10D class an_identifier
    2tired 'c' "Vinny" 2,345 0xFFFF
    Solution

  2. Our Rectangle class from Chapter 1 has instance variables of type int. However, there is no good reason why the dimensions of a rectangle should be whole numbers. Rewrite class Rectangle to have attributes of type double. Now rewrite Program 3.7, the program to calculate the area and perimeter of a rectangle, to use your new class.
    Solution

  3. Using your Rectangle class from Question 2 above, write a Java application that creates three different rectangles with dimensions specified by the user and then prints out the area of each.
    Solution

  4. Declare a Java class whose instances represent books. Such a class might include instance variables giving the title of the book, its authors, the publisher, and the date of publication, and provide methods including those to obtain the title, authors, publisher and date of publication from a book. Use your class in a program that creates a number of books whose details are provided by the user and then prints out the titles of each.
    Solution

  5. Using the types int, double, and String, as well as any types that you define yourself, give a declaration for class Person whose instances represent people. Include at least ten attributes and any methods that seem appropriate. Use your class in an application that creates two instances of Person (representing a male and a female) and makes each the spouse of the other. Your program should then invoke each object to find out its spouse and, having done so, should print out the name of the spouse.

    Suggestion: you might want to consider defining additional classes such as one whose instances represent the person's home or business address.

    Hint: the question should give you clues as to some of the attributes and methods that you should provide.

    Solution

  6. Write a Java application that will read the start time of some event (in hours and minutes) and its duration (also in hours and minutes), and then print out the time at which the event will finish. For example, an event starting at 11:45hrs and lasting 2 hours, 37 minutes finishes at 14:22hrs.

    Hint: think about what classes are required and make sure you consider all the possibilities that might arise when adding a duration to a time.

    Solution

  7. Program 3.14 swaps the value of two variables using a third variable to temporarily remember the value of one of the variables to be swapped. Write a fragment of code that swaps the values of two int variables without using a third variable.
    Solution

  8. A complex number consists of a real part and an imaginary part where the imaginary part is the coefficient of i. (iota - the square root of -1). Write a class whose instances represent complex numbers, and which provides methods to add, subtract, multiply, and divide complex numbers. Use your class in a program to that allows its users to perform each of these operations on two complex numbers that they enter.
    Solution

  9. Write a class whose instances represent points in two-dimensional space and that provides methods to return the x and y co-ordinates of a point and to move a point to a new position.

    Hint: class PointProgram shows how such a class might be used in a program.

    Solution

  10. Trace the execution of PointProgram. In doing so,
    • draw a series of diagrams showing the contents of each of the variables and objects used by the program at each of the points labelled (A)-(G), and
    • say what will be printed on the screen by each of the println statements labelled (1)-(6).
    Solution

  11. Write a Java application that will read two distances, one expressed in yards, feet, and inches and the other expressed in metres, and which will report the sum of the distances as yards, feet, and inches and the difference between the distances as metres.

    Hint: 1 inch is 2.5cm.
