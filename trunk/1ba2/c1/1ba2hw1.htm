Exercises

Chapter 1: Objects and classes

  1. List five classes of object that might be found in each of the following programs:
    • a supermarket stock control program;
    • an air traffic control program;
    • a hospital management program.
    Solution

  2. List (some of) the attributes of instances of each of the classes that you identified in question 1. In each case describe what the type of the attribute is (i.e., what kind of values does it store?)
    Solution

  3. List (some of) the methods that might be provided by each of the classes that you identified in question 1.
    Solution

  4. Modify Program 1.2 so that it prints out a message describing the purpose of the program on the screen before printing out the area and perimeter of the rectangle.
    Solution

  5. Modify Program 1.2 so that it creates two different rectangles with different dimensions and prints out the areas and perimeter of each rectangle in turn.
    Solution

  6. Modify your application from question 5, so that is first prints the areas of both rectangles and then prints the perimeters of the two rectangles.
    Solution

  7. Write an application that creates ten different rectangles and prints out the area and perimeter of each. Your application should not use any more local variables than Program 1.2.
    Solution

  8. Write a class whose instances represent squares and which provides methods to calculate the area and perimeter of a square. Use your class in an application that prints out the area and perimeter of some square.
    Solution

  9. Write a Java application that creates a rectangle and a square and then prints out the area of each.
    Solution

  10. Write a class whose instances represent cubes and which provides methods to calculate the area of one side of the cube, to return the volume of the cube, and any other methods that you think appropriate.
    Solution

  11. Write a Java application that first creates two different cubes and then prints out the areas of their sides and their volumes.
    Solution

  12. Write a class whose instances represent cuboids and that provides methods to calculate the total perimeter of the cuboid, to calculate the volume of the cuboid, to calculate the areas of each of the sides of the cuboid and any other methods that you think appropriate. Use your class in an application that creates at least three objects representing different cuboids and prints out the perimeter and volume of each cuboid as well as the areas of each of their sides.
    Solution
